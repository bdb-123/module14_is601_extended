# Screenshots Directory

This directory contains screenshots demonstrating the CarCompare feature functionality.

## Required Screenshots

Please add the following screenshots to document the CarCompare feature:

1. **browse-cars.png** - List view showing all user's cars in the garage
2. **add-car.png** - Form to add a new car (showing VIN decoder option)
3. **edit-car.png** - Form to update car details
4. **delete-car.png** - Confirmation dialog for car deletion
5. **browse-listings.png** - Table showing all listings for a specific car
6. **add-listing.png** - Form to add a new listing for a car
7. **edit-listing.png** - Form to update listing information (price, mileage, source)
8. **delete-listing.png** - Confirmation dialog for listing deletion
9. **compare-view.png** - Statistics dashboard showing:
   - Total listing count
   - Minimum price
   - Maximum price
   - Average price
   - Average price per mile
   - Best deal identification (listing with lowest price/mile)

## Screenshot Guidelines

- Use a consistent browser and window size
- Include actual data (not just empty states)
- Show the full interface including navigation
- Capture any validation messages or success notifications
- For the compare view, include at least 3-4 listings to demonstrate statistics

## File Naming Convention

Use lowercase with hyphens:
- ✅ `browse-cars.png`
- ❌ `Browse Cars.png`
- ❌ `browse_cars.png`

## Image Format

- Format: PNG
- Resolution: At least 1280x720 (HD)
- File size: Keep under 500KB if possible (compress if needed)
